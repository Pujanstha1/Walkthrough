# DevOps Resources & Templates

A collection of ready-to-use CloudFormation templates, installation guides, and configuration files for common DevOps tasks.

## Contents

- **AWS CloudFormation Template - VPC with Public EC2 Instance** - Standard VPC setup with a publicly accessible EC2 instance
- **Node Exporter Installation Guide** - EC2 instance configured for Prometheus monitoring


### Documentation
- Detailed README files for each template
- Architecture diagrams and component relationships
- Step-by-step deployment instructions

## Usage

Each directory contains:
- Template (`.yaml`)
- README with deployment instructions
- Configuration examples

Simply choose what you need, follow the README, and deploy.

## Quick Start

1. Browse the relevant directory
2. Review the README for prerequisites
3. Deploy using AWS Console or CLI
4. Follow post-deployment configuration steps
